## 推荐更新策略

### 阶段1: 安全更新 (立即执行)
```bash
# 检查安全更新
sudo apt update
sudo apt list --upgradable

# 仅安装安全更新
sudo apt upgrade --only-upgrade
```

### 阶段2: 软件包更新 (谨慎执行)
```bash
# 查看所有可更新软件包
apt list --upgradable

# 选择性更新非关键软件
# 避免更新: nodejs, npm, 核心开发工具
```

### 阶段3: 清理和维护 (最后执行)
```bash
# 清理旧内核和缓存
sudo apt autoremove
sudo apt autoclean

# 检查系统健康
sudo systemctl --failed
journalctl -p 3 -xb
```
