quota-proxy 管理界面

使用说明： 此界面仅供管理员在服务器本机使用（127.0.0.1:8787）。请勿暴露到公网。

1. 配置

2. 创建试用密钥

3. 查看使用情况

4. 重置密钥用量

5. 删除密钥

快速命令参考

# 健康检查 curl -fsS http://127.0.0.1:8787/healthz # 创建密钥（需要 ADMIN_TOKEN） curl -X POST http://127.0.0.1:8787/admin/keys \ -H "Authorization: Bearer $ADMIN_TOKEN" \ -H "Content-Type: application/json" \ -d '{"label":"测试用户"}' # 查看用量 curl -H "Authorization: Bearer $ADMIN_TOKEN" \ http://127.0.0.1:8787/admin/usage # 重置用量 curl -X POST http://127.0.0.1:8787/admin/usage/reset \ -H "Authorization: Bearer $ADMIN_TOKEN" \ -H "Content-Type: application/json" \ -d '{"key":"trial_..."}' # 删除密钥 curl -X DELETE http://127.0.0.1:8787/admin/keys/trial_... \ -H "Authorization: Bearer $ADMIN_TOKEN"
