# 新手一条龙教程（超小白版）

> 目标：从 0 到能用 —— 访问官网、申请/领取 TRIAL_KEY、用 curl 验证额度服务、常见问题排查。

- 网页版（更适合直接复制命令）：https://clawdrepublic.cn/quickstart.html
- quota-proxy 说明页（含接口/错误码/示例）：https://clawdrepublic.cn/quota-proxy.html
- 服务状态页：https://clawdrepublic.cn/status.html

## 1. 你需要准备什么？
- 一台能上网的电脑
- 一个终端（macOS/Linux 用 Terminal；Windows 用 PowerShell）

## 2. 第一步：确认官网可访问
```bash
curl -fsS -m 5 https://clawdrepublic.cn/ > /dev/null && echo OK
```

## 3. 第二步：确认 API 健康
```bash
curl -fsS -m 5 https://api.clawdrepublic.cn/healthz
```

## 4. 第三步：获取 TRIAL_KEY（当前为手动发放）
- 说明：目前 TRIAL_KEY 由管理员手动发放。
- 你需要提供：你的用途/预计调用量/联系方式。

## 5. 第四步：用 TRIAL_KEY 调用 quota-proxy（示例）
> 将下面的 $TRIAL_KEY 替换为你拿到的真实 Key。

建议先把 Key 存成环境变量（后续命令可直接复用）：

- macOS/Linux（bash/zsh）
```bash
export CLAWD_TRIAL_KEY='替换为你的key'
```

- Windows PowerShell
```powershell
$env:CLAWD_TRIAL_KEY='替换为你的key'
```

然后发起请求：
```bash
curl -fsS "https://api.clawdrepublic.cn/usage" -H "Authorization: Bearer $CLAWD_TRIAL_KEY"
```

## 6. 常见问题
- curl 超时：检查网络/代理；稍后重试
- 401/403：Key 不对或未开通
- 5xx：服务端异常，建议带上请求时间与返回内容反馈
