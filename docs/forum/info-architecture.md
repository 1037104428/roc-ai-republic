## 部署方案（MVP）

### 阶段1：静态页面
- 使用 GitHub Pages + Markdown
- 手动更新帖子（通过 Git 提交）
- 评论使用 GitHub Issues 或外部服务（如 utterances）

### 阶段2：简单动态
- 使用轻量级论坛引擎（如 Discourse、Flarum）
- 部署在 VPS 或容器中
- 基础用户系统 + 发帖功能

### 阶段3：完整功能
- 用户积分/等级系统
- 私信功能
- 高级搜索
- 移动端优化
