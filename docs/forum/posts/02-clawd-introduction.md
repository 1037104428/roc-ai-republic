# Clawd 国度介绍

欢迎来到 Clawd 国度！这是一个专注于 AI 助手开发和部署的开源社区。

## 什么是 Clawd 国度？

Clawd 国度是一个围绕 OpenClaw 项目构建的生态系统，旨在：

1. **降低 AI 助手开发门槛**：提供简单易用的工具和框架
2. **构建开源 AI 助手生态**：鼓励协作和知识共享
3. **推动 AI 助手实际应用**：从个人助手到企业级解决方案

## 核心项目

### 1. OpenClaw
- **描述**：开源 AI 助手框架，支持多平台、多模型
- **特点**：模块化设计、易于扩展、社区驱动
- **GitHub**：https://github.com/openclaw/openclaw

### 2. quota-proxy
- **描述**：AI API 配额管理和代理服务
- **特点**：多模型支持、使用量统计、费用控制
- **用途**：为开发者提供稳定的 AI 服务接入

### 3. ClawHub
- **描述**：技能市场，分享和获取 AI 助手技能
- **特点**：一键安装、版本管理、社区评分
- **网址**：https://clawhub.com

## 如何开始？

### 新手入门
1. **阅读快速入门**：https://clawdrepublic.cn/quickstart.html
2. **申请 TRIAL_KEY**：在 quota-proxy 页面申请测试密钥
3. **安装 OpenClaw**：按照文档完成安装配置
4. **加入社区**：参与论坛讨论，关注最新动态

### 开发者参与
1. **熟悉代码**：阅读项目源码和架构文档
2. **选择任务**：从 Good First Issue 开始
3. **提交贡献**：按照贡献指南提交 PR
4. **持续参与**：成为核心贡献者
