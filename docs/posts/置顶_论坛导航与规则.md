# （置顶）Clawd 国度论坛导航与规则

欢迎来到 Clawd 国度。

## 你来这里通常是为了解决三件事

1) **新手跑通（推荐从这里开始）**
- 小白一条龙：<https://clawdrepublic.cn/quickstart.html>

2) **申请 / 领取 TRIAL_KEY（试用额度）**
- 说明页（含最小 curl 验证）：<https://clawdrepublic.cn/quota-proxy.html>
- 发帖板块：TRIAL_KEY 申请
- 发帖时请照抄置顶模板（见下方）

3) **报错/故障排查**
- 去「问题求助」板块
- 发帖时请照抄模板（见下方），把关键日志贴出来

## 发帖基本规则（为了让大家更快帮到你）

- 标题要可检索：带上关键词（OpenClaw / quota-proxy / 401 / 429 / 安装失败等）
- 不要公开粘贴完整 key：
  - TRIAL_KEY 只在私信/工单里发
  - 如必须展示，请脱敏（例如：`sk-abcd...wxyz`）
- 你做过的尝试请写出来：别人就不会重复问你“你试过了吗”

## 模板帖

- TRIAL_KEY 申请模板：见《模板_TRIAL_KEY_申请》
- 问题求助模板：见《模板_问题求助》

（管理员）建议把这两篇模板帖也置顶到对应板块。
