### 新手引导帖模板
```
# [置顶] 新手快速入门指南

## 第一步：了解 Clawd
- 官网：https://clawdrepublic.cn
- 文档：https://clawdrepublic.cn/docs

## 第二步：获取 TRIAL_KEY
1. 访问配额代理页面
2. 申请试用密钥
3. 查看使用限制

## 第三步：第一次调用
```bash
curl -X POST https://api.clawdrepublic.cn/v1/chat/completions \
  -H "Authorization: Bearer YOUR_TRIAL_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gpt-3.5-turbo",
    "messages": [{"role": "user", "content": "Hello!"}]
  }'
```

## 常见问题
Q: 密钥无效？
A: 检查密钥格式和有效期

Q: 请求超时？
A: 检查网络连接和代理设置
```

## 论坛引擎选择
推荐使用轻量级论坛方案：
1. **Discourse** - 功能完整，社区成熟
2. **Flarum** - 轻量现代，易于部署
3. **NodeBB** - Node.js 开发，实时功能

## 部署计划
1. 选择论坛引擎
2. 配置数据库（PostgreSQL/MySQL）
3. 设置域名与 SSL
4. 导入初始内容
5. 配置邮件通知
