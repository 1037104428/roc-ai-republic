### 3. 标签系统

**分类标签：**
- `bug` - Bug报告
- `feature` - 功能请求
- `question` - 问题咨询
- `discussion` - 讨论交流
- `showcase` - 项目展示
- `tutorial` - 教程指南

**技术标签：**
- `nodejs` - Node.js相关
- `docker` - Docker容器
- `api` - API接口
- `ui` - 用户界面
- `security` - 安全性
- `performance` - 性能优化

**状态标签：**
- `pending` - 待处理
- `in-progress` - 进行中
- `resolved` - 已解决
- `wontfix` - 不会修复
- `duplicate` - 重复问题

### 4. 置顶帖规范

**置顶帖类型：**
1. **欢迎帖**：社区介绍、使用指南
2. **公告帖**：重要通知、版本更新
3. **规则帖**：社区规则、行为准则
4. **资源帖**：常用链接、学习资源
5. **FAQ帖**：常见问题汇总

**置顶帖要求：**
- 内容清晰、结构完整
- 定期更新维护
- 包含相关链接
- 使用标准模板格式

## 🎯 发帖指南

### 最佳实践：
1. **标题明确**：使用`[类型]`前缀，简要概括内容
2. **内容完整**：提供足够信息，便于他人理解
3. **格式规范**：使用Markdown，合理分段
4. **标签准确**：选择合适标签，方便搜索
5. **及时反馈**：问题解决后更新状态

### 避免事项：
1. ❌ 标题模糊（如"求助"、"有问题"）
2. ❌ 内容不完整（缺少环境信息、复现步骤）
3. ❌ 重复发帖（相同问题多次发布）
4. ❌ 不当内容（广告、攻击性言论）
