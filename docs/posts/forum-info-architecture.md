# 论坛信息架构与内容规划

## 论坛定位
- **名称**: Clawd 共和国论坛
- **目标**: 为 Clawd 用户提供交流、支持、反馈的平台
- **技术栈**: NodeBB (Node.js + Redis/SQLite)

## 核心板块结构

### 1. 公告区 (Announcements)
- **置顶帖**: 论坛规则、使用指南
- **官方公告**: 版本更新、服务状态、重要通知

### 2. 新手入门 (Getting Started)
- **置顶帖**: 快速上手指南、常见问题
- **模板帖**: "我是新手，请问..."、"如何获取 TRIAL_KEY"

### 3. 技术支持 (Technical Support)
- **分类**: 
  - API 使用问题
  - 部署与配置
  - 故障排查
- **模板帖**: "错误报告模板"、"功能请求模板"

### 4. 项目展示 (Showcase)
- 用户项目分享
- 最佳实践案例

### 5. 社区讨论 (Community)
- 功能建议
- 技术交流
- 水区闲聊

#### 2. 快速上手指南
```
# Clawd 快速上手指南

## 1. 获取 TRIAL_KEY
访问 https://clawdrepublic.cn/quota-proxy.html 了解试用密钥获取方式

## 2. 首次 API 调用
```bash
curl -X POST https://api.clawdrepublic.cn/v1/chat/completions \
  -H "Authorization: Bearer YOUR_TRIAL_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "claude-3-haiku",
    "messages": [{"role": "user", "content": "Hello"}]
  }'
```

## 3. 常见问题
- Q: 如何查看使用量？
- A: 访问 https://api.clawdrepublic.cn/admin/usage (需要管理员权限)
```
