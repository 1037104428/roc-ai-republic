# 论坛内容管理指南

## 概述
本指南为 Clawd 国度论坛管理员和版主提供内容管理的最佳实践和操作流程。

## 内容分类

### 1. 置顶帖（Sticky Posts）
- **欢迎帖**：新用户引导，社区规则介绍
- **公告帖**：重要通知，版本更新，活动公告
- **FAQ帖**：常见问题解答
- **资源索引帖**：重要资源链接汇总

### 2. 常规帖子
- **技术讨论**：AI开发、编程、部署问题
- **项目展示**：用户项目分享
- **教程分享**：技术教程、使用指南
- **问题求助**：技术问题求助
- **意见建议**：社区改进建议

### 3. 模板帖子
- **问题求助模板**：规范问题描述格式
- **项目展示模板**：统一项目介绍格式
- **教程分享模板**：标准化教程结构

## 内容管理流程

### 1. 发帖审核
```
新帖子 → 自动审核（关键词过滤） → 人工审核（可选） → 发布
```

### 2. 内容维护
- **每日检查**：新帖子审核、垃圾内容清理
- **每周整理**：精华帖归档、过时内容归档
- **每月总结**：热门话题分析、内容质量评估

### 3. 用户管理
- **新用户引导**：欢迎消息、新手任务
- **活跃用户激励**：积分奖励、徽章系统
- **违规处理**：警告、禁言、封禁流程

## 最佳实践

### 1. 内容质量
- 标题清晰明确
- 内容结构清晰
- 代码使用代码块
- 图片添加描述

### 2. 社区互动
- 及时回复用户问题
- 鼓励建设性讨论
- 处理争议时保持中立
- 尊重不同观点

### 3. 安全管理
- 定期备份内容
- 监控异常活动
- 及时处理违规内容
- 保护用户隐私

## 工具和脚本

### 1. 管理脚本
- `init-forum-sticky-posts.sh`：初始化置顶帖
- `check-forum-service.sh`：检查论坛服务状态
- `verify-forum-access.sh`：验证论坛访问

### 2. 监控工具
- 服务状态监控
- 内容质量分析
- 用户活跃度统计

## 紧急处理

### 1. 服务故障
1. 检查服务状态：`./scripts/check-forum-service.sh`
2. 查看错误日志：`docker logs forum`
3. 重启服务：`docker restart forum`

### 2. 内容安全事件
1. 立即删除违规内容
2. 记录事件详情
3. 通知相关用户
4. 更新安全规则
