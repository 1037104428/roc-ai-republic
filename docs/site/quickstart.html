Clawd 国度 · OpenClaw 小白一条龙（免翻墙）

返回首页 · 进入论坛 · GitHub

你将获得什么

  • 国内可直连安装 OpenClaw
  • 默认使用 Clawd 国度的 DeepSeek 限额网关（不用自己申请 DeepSeek key）

第 0 步：先准备 Node.js（如果你已经有 npm，可跳过）

OpenClaw 依赖 Node.js。若你不确定有没有：在终端输入 npm -v，能输出版本号就行。

第 1 步：一条命令安装 OpenClaw（国内源优先）

curl -fsSL https://clawdrepublic.cn/install-cn.sh | bash

安装完成后，验证：

openclaw --version

第 2 步：写入配置（复制粘贴即可）

把下面内容保存为：~/.openclaw/openclaw.json

{
  "agents": {
    "defaults": {
      "model": { "primary": "clawd-gateway/deepseek-chat" },
      "models": {
        "clawd-gateway/deepseek-chat": {},
        "clawd-gateway/deepseek-reasoner": {}
      }
    }
  },
  "models": {
    "mode": "merge",
    "providers": {
      "clawd-gateway": {
        "baseUrl": "https://api.clawdrepublic.cn/v1",
        "apiKey": "${CLAWD_TRIAL_KEY}",
        "api": "openai-completions",
        "models": [
          { "id": "deepseek-chat", "name": "DeepSeek Chat" },
          { "id": "deepseek-reasoner", "name": "DeepSeek Reasoner" }
        ]
      }
    }
  }
}

第 3 步：拿到试用额度（TRIAL_KEY）

你需要一个 CLAWD_TRIAL_KEY（试用 key）。目前先走“人工发放”，避免被滥用：

  1. 去论坛发一帖：/forum/（标题写：申请 TRIAL_KEY）
  2. 把你要做的事写清楚（1-2 句话即可）

拿到 key 后，在终端执行（把 xxx 替换成你的 key）：

export CLAWD_TRIAL_KEY="xxx"

第 4 步：先用 curl 做一次“最小验证”（推荐）

这一步可以帮你快速确认：key 是对的、API 可用、网络通。

curl -fsS https://api.clawdrepublic.cn/healthz
curl -fsS https://api.clawdrepublic.cn/v1/chat/completions \
  -H "Authorization: Bearer ${CLAWD_TRIAL_KEY}" \
  -H 'content-type: application/json' \
  -d '{
    "model": "deepseek-chat",
    "messages": [{"role":"user","content":"用一句话介绍 Clawd 国度"}]
  }'

第 5 步：启动 OpenClaw 网关并验证

openclaw gateway start
openclaw models status

如果一切正常，你会看到默认模型指向 clawd-gateway/...。然后就可以开始用 OpenClaw 了。

API 健康检查：https://api.clawdrepublic.cn/healthz
