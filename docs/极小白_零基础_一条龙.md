# 🐾 极小白零基础一条龙教程

> 写给：完全不懂命令行、第一次接触AI API的小白用户
> 目标：复制粘贴就能跑通，不需要任何技术背景

## 📱 手机用户请先看这里

如果你只有手机，请先看：[手机版零基础3分钟上手教程](./tutorials/手机版_零基础_3分钟上手.md)

## 💻 电脑用户（Windows/Mac/Linux）从这里开始

### 第一步：打开"黑框框"（终端）

- **Windows**：按 `Win + R`，输入 `powershell`，回车
- **Mac**：按 `Cmd + 空格`，输入 `terminal`，回车
- **Linux**：按 `Ctrl + Alt + T`

### 第二步：测试网络连接（不花钱）

复制下面这行，粘贴到黑框框里，按回车：

```bash
curl -fsS -m 5 https://clawdrepublic.cn/ > /dev/null && echo "✅ 网络连接正常"
```

如果看到 `✅ 网络连接正常`，恭喜！继续下一步。

如果看到错误，请截图保存，然后去论坛求助。

### 第三步：测试API健康（不花钱）

复制粘贴：

```bash
curl -fsS -m 5 https://api.clawdrepublic.cn/healthz
```

应该看到：
```json
{"ok":true}
```

### 第四步：一键全面检查（推荐）

复制粘贴这个命令，它会自动检查所有项目：

```bash
curl -fsSL https://clawdrepublic.cn/probe-roc-all.sh | bash
```

等待几秒钟，你会看到类似这样的结果：
```
✅ web_ok
✅ api_healthz_ok
✅ api_models_ok
✅ quota_proxy_ok
```

### 第五步：申请试用密钥（需要人工审核）

1. 打开浏览器访问：https://clawdrepublic.cn/forum/
2. 注册账号（用邮箱）
3. 在"问题求助"板块发帖，标题写："申请试用密钥"
4. 内容写：
   - 用途：学习/测试/项目开发
   - 预计每天使用次数：10-20次
   - 想用的模型：GPT-4或Claude（不确定就写"不确定"）

管理员会在24小时内通过私信或回复给你一个密钥，格式类似：`sk-trial_xxxx`

### 第六步：设置密钥（一次性）

拿到密钥后，在黑框框里输入（把 `sk-trial_xxxx` 换成你的真实密钥）：

```bash
export CLAWD_TRIAL_KEY='sk-trial_xxxx'
```

**Windows用户注意**：用这个命令：
```powershell
$env:CLAWD_TRIAL_KEY='sk-trial_xxxx'
```

### 第七步：第一次调用AI（会消耗额度）

复制粘贴这个命令：

```bash
curl -X POST https://api.clawdrepublic.cn/v1/chat/completions \
  -H "Authorization: Bearer $CLAWD_TRIAL_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gpt-4",
    "messages": [
      {"role": "user", "content": "你好，请用一句话介绍你自己"}
    ],
    "max_tokens": 50
  }'
```

如果成功，你会看到一大段JSON，里面有AI的回答。

### 🎉 恭喜！你已经成功调用了AI！

## ❓ 常见问题

### Q: 密钥有效期多久？
A: 试用密钥通常有7-30天有效期，具体看管理员设置。

### Q: 额度用完了怎么办？
A: 可以去论坛申请增加额度，说明使用情况和需求。

### Q: 可以商用吗？
A: 试用密钥仅供测试，商用需要联系管理员获取正式授权。

### Q: 支持哪些模型？
A: 支持GPT-4、Claude、DeepSeek等主流模型，具体看[模型列表](../api/models.md)。

### Q: 速度慢怎么办？
A: 可能是网络问题，可以尝试：
   - 换个时间段再试
   - 检查本地网络
   - 去论坛反馈
