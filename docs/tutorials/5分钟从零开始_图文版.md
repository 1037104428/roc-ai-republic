# 5分钟从零开始：图文版 Clawd AI 使用指南

## 📋 准备工作

### 1. 申请 TRIAL_KEY
- 打开浏览器，访问：https://clawdrepublic.cn
- 点击"申请试用"按钮
- 填写简单信息：姓名、邮箱、用途
- 提交后等待管理员审核（通常24小时内）

### 2. 拿到密钥后
你会收到一封邮件，里面包含：
```
你的 TRIAL_KEY: trial_xxxxxxxxxxxxxxxx
有效期：30天
每日限额：100次调用
```

## 🖥️ Windows 用户（完全不懂命令行）

### 方法一：使用在线工具（最简单）
1. 打开浏览器访问：https://reqbin.com/
2. 在左侧选择 `POST` 方法
3. 在 URL 输入框粘贴：`https://api.clawdrepublic.cn/v1/chat/completions`
4. 点击 `Headers` 标签，添加：
   - `Authorization`: `Bearer trial_你的密钥`
   - `Content-Type`: `application/json`
5. 点击 `Body` 标签，选择 `Raw`，粘贴：
```json
{
  "model": "clawd-default",
  "messages": [{"role": "user", "content": "你好！"}]
}
```
6. 点击 `Send` 按钮
7. 在右侧查看结果

### 方法二：使用 PowerShell（推荐）
1. 按 `Win + R`，输入 `powershell`，回车
2. 复制下面命令，把 `YOUR_KEY` 换成你的密钥：
```powershell
$key = "trial_你的密钥"
$body = @{
    model = "clawd-default"
    messages = @(
        @{role = "user"; content = "你好！"}
    )
} | ConvertTo-Json

Invoke-RestMethod -Uri "https://api.clawdrepublic.cn/v1/chat/completions" `
    -Method Post `
    -Headers @{
        "Authorization" = "Bearer $key"
        "Content-Type" = "application/json"
    } `
    -Body $body
```
3. 粘贴到 PowerShell，回车
4. 看到返回结果就成功了！

## 🍎 Mac 用户

### 使用终端（Terminal）
1. 按 `Command + 空格`，输入 `terminal`，回车
2. 复制下面命令，把 `YOUR_KEY` 换成你的密钥：
```bash
curl -sS https://api.clawdrepublic.cn/v1/chat/completions \
  -H "Authorization: Bearer trial_你的密钥" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "clawd-default",
    "messages": [{"role": "user", "content": "你好！"}]
  }'
```
3. 粘贴到终端，回车
4. 看到返回结果就成功了！

## 📱 手机用户

### 使用"API测试"APP
1. 在应用商店搜索 `API Tester` 或 `Postman` 下载
2. 打开APP，新建一个请求
3. 设置：
   - 方法：`POST`
   - URL：`https://api.clawdrepublic.cn/v1/chat/completions`
   - Headers：
     - `Authorization`: `Bearer trial_你的密钥`
     - `Content-Type`: `application/json`
   - Body（选择JSON）：
```json
{
  "model": "clawd-default",
  "messages": [{"role": "user", "content": "你好！"}]
}
```
4. 点击发送
5. 查看响应

## ❓ 常见问题解答

### Q1：我完全不懂编程，能用吗？
**能！** 按照上面的图文步骤，不需要懂编程。

### Q2：密钥输错了怎么办？
检查：
1. 有没有多复制空格
2. 有没有少复制字符
3. `Bearer` 后面要有一个空格

### Q3：为什么没反应？
可能原因：
1. 网络问题：检查是否能访问 `api.clawdrepublic.cn`
2. 密钥过期：联系管理员
3. 超过每日限额：明天再试

### Q4：能问什么问题？
任何问题！比如：
- "帮我写一个生日祝福语"
- "解释什么是人工智能"
- "推荐几本好书"
- "帮我规划周末行程"

## 🎁 小技巧

### 1. 保存命令到文件
把命令保存为 `ask.bat`（Windows）或 `ask.sh`（Mac/Linux），下次双击就能用。

### 2. 问不同问题
只需要改 `content` 里的文字：
```json
"content": "帮我写一个Python Hello World程序"
```

### 3. 连续对话
把之前的对话也加进去：
```json
"messages": [
  {"role": "user", "content": "什么是AI？"},
  {"role": "assistant", "content": "AI是人工智能..."},
  {"role": "user", "content": "那机器学习呢？"}
]
```
