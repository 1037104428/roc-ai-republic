# 极小白：30秒开始使用 Clawd AI

## 🚀 30秒快速开始

1. **拿到密钥**：向管理员申请 `TRIAL_KEY`
2. **复制命令**：把下面的 `YOUR_KEY` 换成你的密钥
3. **运行**：在终端里粘贴运行

```bash
curl -sS https://api.clawdrepublic.cn/v1/chat/completions \
  -H "Authorization: Bearer YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "clawd-default",
    "messages": [{"role": "user", "content": "你好！"}]
  }'
```

✅ 看到返回结果就成功了！

## 📦 你需要什么？

- 一个 TRIAL_KEY（找管理员要）
- 能上网的电脑
- 终端（Mac/Linux 用 Terminal，Windows 用 Git Bash）

## 🔑 如何拿到密钥？

**当前方式**：找管理员申请（后续会有自助申请）

**申请信息**：
- 用途：学习/测试/项目
- 预计每天用几次
- 联系方式（可选）

**申请渠道**：
- GitHub Issues
- 社区论坛
- 邮件：admin@clawdrepublic.cn

## 📊 查看剩余次数

```bash
curl -sS https://api.clawdrepublic.cn/admin/usage \
  -H "Authorization: Bearer YOUR_KEY"
```

## ❓ 常见问题

**Q：密钥长什么样？**
A：类似 `trial_7x9y2z8w5v4u3t6s1r0q`

**Q：能用多久？**
A：30天，每天最多100次调用

**Q：密钥泄露了怎么办？**
A：立即联系管理员，我们会停用旧密钥发新密钥

**Q：支持哪些模型？**
A：目前支持 `clawd-default`（DeepSeek 兼容）

## 🆘 遇到问题？

1. **检查密钥**：确认密钥正确，没有空格
2. **检查网络**：确保能访问 `api.clawdrepublic.cn`
3. **查看错误**：命令会显示错误信息
4. **联系支持**：support@clawdrepublic.cn
