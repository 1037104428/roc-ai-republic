OpenClaw 小白版（免翻墙）

v0 安装脚本（国内 npm 源优先）：

curl -fsSL https://clawdrepublic.cn/install-cn.sh | bash

默认模型（限额试用）

DeepSeek 限额转发（OpenAI-compatible）：

Base URL: https://api.clawdrepublic.cn/v1
Auth:     Authorization: Bearer <TRIAL_KEY>

TRIAL_KEY 获取与使用（当前：手动发放）

我们处于早期试运行阶段：TRIAL_KEY 由管理员手动签发（后续会上线自助申请/自动签发）。

  1. 准备一段申请信息（复制粘贴即可）：用途 / 预计试用时长 / 预计每天请求量 / 你要用的模型（deepseek-chat 或 deepseek-reasoner）。
  2. 把申请信息发给管理员（社区/群/私信渠道）。
  3. 收到 TRIAL_KEY 后，按下面最小命令验证。

提示：不要把 key 贴到公开地方；被别人用光额度，你自己就用不了了。

最小可验证（curl 直连网关）

# 1) 不产生调用成本：只测网关健康
curl -fsS https://api.clawdrepublic.cn/healthz

# 2) 产生调用成本：最小一次对话（把 xxx 换成你拿到的 TRIAL_KEY）
export TRIAL_KEY="xxx"

curl -fsS https://api.clawdrepublic.cn/v1/chat/completions \
  -H "Authorization: Bearer ${TRIAL_KEY}" \
  -H 'content-type: application/json' \
  -d '{
    "model": "deepseek-chat",
    "messages": [{"role":"user","content":"用一句话介绍 Clawd 国度"}]
  }'

（可选）设置成 OpenAI 兼容环境变量

export OPENAI_API_KEY="${TRIAL_KEY}"
export OPENAI_BASE_URL="https://api.clawdrepublic.cn/v1"
