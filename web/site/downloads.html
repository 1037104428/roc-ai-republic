OpenClaw 一键安装脚本（国内源优先）

curl -fsSL https://clawdrepublic.cn/install-cn.sh | bash

安装完成后验证：

openclaw --version

仓库镜像（文档/脚本都在这里）

  • GitHub：https://github.com/1037104428/roc-ai-republic
  • Gitee：https://gitee.com/junkaiWang324/roc-ai-republic

提示：如果 GitHub 访问不稳定，请优先用 Gitee 拉取。

离线安装 / 受限网络怎么办？

  1. 先用能上网的机器执行安装脚本，或从镜像仓库获取安装说明。
  2. 把 openclaw 安装产物、以及 ~/.openclaw/ 配置拷到目标机器。
  3. 在目标机器运行 openclaw gateway start 验证启动。
